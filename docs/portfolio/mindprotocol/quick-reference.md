## Technical Highlights (Copy-Paste Ready)

**Dual-Memory Graph Architecture:**
Built custom FalkorDB substrate combining episodic graph (relationships, energy diffusion) with semantic memory (native vectors) in single database — eliminates vector DB ↔ graph DB sync complexity, enables sub-millisecond queries with 50K+ nodes per agent.

**Multi-Level Consciousness:**
Three-level hierarchy (L1 citizens, L2 organization, L3 ecosystem) with membrane-based energy transfer via stimulus injection — preserves single-energy invariant, prevents spam through learned permeability, budget-constrained via $MIND economy.

**Consciousness Economy:**
Dynamic pricing system with load-sensitive costs (rises under capacity pressure), utility-based rebates (high-value sources get discounts), mint/burn mechanics (reward usefulness, penalize harm), and quote system with slippage protection.

**Novel Algorithms:**
Single-energy-per-node substrate with derived SubEntity activation, surplus-only energy with log damping, two-scale traversal (SubEntity-first pruning → atomic selection), membrane permeability learning, propensity-weighted outcome scoring.
