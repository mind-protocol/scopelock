## Key ScopeLock Concepts (Explained Simply)

### AC.md (Acceptance Criteria)
**What:** A document listing exactly what the software must do to be "done."
**Why:** Prevents disputes - client and team agree on requirements BEFORE coding.
**Example:** "User can log in with email + password in <2 seconds."
**Format:** Functional criteria (features work) + Non-functional (performance, quality).

### AC Green
**What:** The moment when all acceptance tests pass.
**Why:** This is when client pays - tests prove the work is done.
**How:** Automated tests run and all show green checkmarks ‚úÖ.

### Evidence Sprint
**What:** A quick demo (‚â§90 seconds) showing working functionality.
**Why:** Proves progress with actual working code, not just promises.
**Includes:** Demo video + DELTA.md (2-3 quantified improvements).
**Timeline:** 6-8 hours of focused work OR 2-5 days depending on scope.

### Proof Log
**What:** Public record of all deliveries at scopelock.mindprotocol.ai.
**Why:** Shows clients our track record and builds trust.
**Contains:** AC.md, demo links, quantified results for each mission.

### Change Control (Swap vs Add)
**What:** How we handle client requests to change scope.

**Swap:** Replace a feature with something equally complex
- Cost: $0 (no price change)
- Example: "Replace login with email with login with Google" (same difficulty)

**Add:** Add a new feature not in original AC.md
- Cost: Priced as new milestone
- Example: "Add admin dashboard" (new functionality)

**Why:** Prevents scope creep while being flexible.

### DoD (Definition of Done) Checklist
**What:** A checklist of everything needed before delivery.
**Why:** Ensures nothing is forgotten.
**Includes:** Tests pass, docs written, deployed, client credentials provided, monitoring configured, proof log entry, etc.

### Mission
**What:** A project or feature set with its own AC.md and price.
**Why:** We break work into missions with clear scope and delivery points.
**Duration:** Usually 5 days to 4 weeks depending on complexity.
