## Executive Summary

**We serve two distinct client personas:**

1. **Business Buyers (Primary)** - Founders, CTOs, marketing managers who need creative work done
   - **What they buy:** Images, videos, music, voice, presentations, translations, content
   - **Why they buy from us:** AI-native workflow = 10x cheaper + faster than traditional agencies
   - **Budget:** $400-1500 per mission
   - **Margins:** 70-85%

2. **Technical Buyers (Secondary)** - Developers, CTOs who need code/integrations
   - **What they buy:** APIs, bots, landing pages, document processing
   - **Why they buy from us:** Fixed price, executable AC, pay only when tests pass
   - **Budget:** $200-600 per mission
   - **Margins:** 50-60%

**Strategic Priority:** Focus on Business Buyers first (higher margins, lower competition), use Technical Buyers as fallback when creative pipeline is empty.

## PRICING STRATEGY BY MISSION TYPE

### Creative AI Services (70-85% Margin)

**Formula:**
```
Base Cost = API Cost + Human Time (QA/Review)
Target Price = Base Cost × 3-5x (depending on complexity)

Example: Gamma Presentation
- API Cost: $15 (Gamma Pro)
- Human Time: 4 hours × $50/hour = $200
- Base Cost: $215
- Target Price: $215 × 3 = $645
- Quoted Price: $600 (rounded)
- Margin: 64%
```

**Pricing Tiers:**
- **Simple:** $300-500 (10-20 deliverables, clear template, 1 revision)
- **Medium:** $500-800 (20-50 deliverables, custom template, 2 revisions)
- **Complex:** $800-1500 (50+ deliverables OR custom creative direction, 2+ revisions)

---

### AI Integration (60-70% Margin)

**Formula:**
```
Base Cost = Developer Time (implementation + testing) + API Cost
Target Price = Base Cost × 2-2.5x

Example: PDF Parsing
- Developer Time: 8 hours × $8/hour = $64
- API Cost: $10 (Claude)
- Overhead: $15 (20%)
- Base Cost: $89
- Target Price: $89 × 6 = $534
- Quoted Price: $500 (rounded)
- Margin: 82% (high because junior dev is cheap)
```

**Pricing Tiers:**
- **Simple:** $300-500 (1-2 days, clear requirements)
- **Medium:** $500-800 (3-5 days, moderate complexity)
- **Complex:** $800-1200 (5-7 days, custom integration)

---

### Simple Dev Work (50-60% Margin)

**Formula:**
```
Base Cost = Developer Time + Overhead
Target Price = Base Cost × 2x

Example: Landing Page
- Developer Time: 6 hours × $8/hour = $48
- Overhead: $10 (20%)
- Base Cost: $58
- Target Price: $58 × 6 = $348
- Quoted Price: $350 (rounded)
- Margin: 83% (junior dev is very cheap)
```

**Pricing Tiers:**
- **Simple:** $200-400 (landing page, simple bot)
- **Medium:** $400-600 (CRUD app, REST API)
